一.java中,散列用链表数组实现.其中每个列表被称之为桶,想要查找表中对象位置,就要先计算他的散列码,
然后与桶的总数取余,所得到的结果就是保存这个元素的桶的索引了.
例: 某对象的散列码为76268,并且有128个桶,对象应该保存在108号桶中(76268%128=108),或许会很幸运这个桶中没有其他元素,
此时将该元素直接放入该桶中即可;但当该桶有元素情况也会存在,该情况称为散列冲突.这时需要将新对象与桶内所有元素逐一比较,
查看该对象是否存在.
